### 唯一标识【ctrlId】

只读属性，不可修改，作为每个组件实例的一个唯一标识

-   标识：`ctrlId`
-   默认值：`@[packageid]`

### 基本属性

##### 数据源接受参数

```js
IDM.datasource.request(this.propData?.dataSource?.[0]?.id, {
    moduleObject: this.moduleObject,
    param: IDM.router.getParam(this.moduleObject.routerId) // 上个页面传过来的参数
})
```
